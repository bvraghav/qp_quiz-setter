2. Create /Question DB/ as CSV file with following
   header:
   1. Header String: Q,A0,A1,A2,A3,A,A_MUL_P
   2. Q: Question Text;
   3. A0: Answer 0 Text;
   4. A1: Answer 1 Text;
   5. A2: Answer 2 Text;
   6. A3: Answer 3 Text;
   7. A: Correct Answer Index {0,1,2,3};
   8. A_MUL_P: Is the answer multi-line {0,1};
3. Create a relative symlink =qdb.csv= to point to the
   /Question DB/
